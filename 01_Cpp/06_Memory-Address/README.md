
# Build and Run via Command line

Build
```Shell
g++ -g 02_Memory-Address/main.cpp -o 02_Memory-Address/main
```

Run
```Shell
./02_Memory-Address/main
```

# Build and Run via VSCode

Build, select task
```Shell
Ctrl + Shift + B
```
Select Task
`build 02_Memory-Address` 

Run
```Shell
F5
```

Must be set right lauch task
