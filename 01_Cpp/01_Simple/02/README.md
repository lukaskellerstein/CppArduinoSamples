
# Build and Run via Command line

Build
```Shell
g++ -g 01_Simple/02/simple.cpp -o 01_Simple/02/simple
```

Run
```Shell
./01_Simple/02/simple
```

# Build and Run via VSCode

Build, select task
```Shell
Ctrl + Shift + B
```
Select Task
`build 01_Simple/02` 

Run
```Shell
F5
```

Must be set the right lauch task
