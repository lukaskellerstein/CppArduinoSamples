
# Build and Run via Command line

Build
```Shell
g++ -g 01/simple.cpp -o 01/simple
```

Run
```Shell
./01/simple
```

# Build and Run via VSCode

Build, select task
```Shell
Ctrl + Shift + B
```

Run
```Shell
F5
```
