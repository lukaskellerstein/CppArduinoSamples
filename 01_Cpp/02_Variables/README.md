
# Build and Run via Command line

Build
```Shell
g++ -g 02_Variables/main.cpp -o 02_Variables/main
```

Run
```Shell
./02_Variables/main
```

# Build and Run via VSCode

Build, select task
```Shell
Ctrl + Shift + B
```
Select Task
`build 02_Variables` 

Run
```Shell
F5
```

Must be set right lauch task
