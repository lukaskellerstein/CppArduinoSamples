
# Build and Run via Command line

Build
```Shell
g++ -g 03_Pointers/01/main.cpp -o 03_Pointers/01/main
```

Run
```Shell
./03_Pointers/01/main
```

# Build and Run via VSCode

Build, select task
```Shell
Ctrl + Shift + B
```
Select Task
`build 03_Pointers/01` 

Run
```Shell
F5
```

Must be set right lauch task
