
# Build and Run via Command line

Build
```Shell
g++ -g 05_Collections/main.cpp -o 05_Collections/main
```

Run
```Shell
./05_Collections/main
```

# Build and Run via VSCode

Build, select task
```Shell
Ctrl + Shift + B
```
Select Task
`build 05_Collections` 

Run
```Shell
F5
```

Must be set right lauch task
