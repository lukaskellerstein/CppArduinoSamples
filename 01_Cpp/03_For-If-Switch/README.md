
# Build and Run via Command line

Build
```Shell
g++ -g 03_For-If-Switch/main.cpp -o 03_For-If-Switch/main
```

Run
```Shell
./03_For-If-Switch/main
```

# Build and Run via VSCode

Build, select task
```Shell
Ctrl + Shift + B
```
Select Task
`build 03_For-If-Switch` 

Run
```Shell
F5
```

Must be set right lauch task
