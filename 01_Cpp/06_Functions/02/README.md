
# Build and Run via Command line

Build
```Shell
g++ -g 06_Functions/02/main.cpp -o 06_Functions/02/main
```

Run
```Shell
./06_Functions/02/main
```

# Build and Run via VSCode

Build, select task
```Shell
Ctrl + Shift + B
```
Select Task
`build 06_Functions/02` 

Run
```Shell
F5
```

Must be set right lauch task
